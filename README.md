# The Mix Website

Baseline Next.js + React + TypeScript framework for **The Mix**, a music networking/business organization at Harvard University.

## Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- ESLint

## Quick start

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Project structure

```text
src/
  app/
    about/
    contact/
    programs/
    globals.css
    layout.tsx
    page.tsx
  components/
    Footer.tsx
    NavBar.tsx
    SectionCard.tsx
  lib/
    site.ts
```
